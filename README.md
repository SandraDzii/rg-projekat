# Implementirane oblasti
* 1-8 nedelja
* face culling
* blending (blend)
* advanced lighting
* A - anti aliasing
* B - normal mapping

# Uputstvo
* F1 - on/off imGui
* F2 - on/off dirlight
* F3 - on/off spotlight sa kamere 
* imGui - on/off cekirani pointlight
